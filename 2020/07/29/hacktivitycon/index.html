Prophecy

This problem provides a connection to guess numbers. Providing a wrong number gives you the expected number and closes the connection.

Tootsie Pop

This problem provides a zip file that turns out to be a single file compressed over and over again using different formats.

Edit: After looking over some other write-ups using 7z would have been a much better solution as it’s able to assume file compression and just extract. I didn’t think of that at the time.

Flag: flag{the_answer_is_1548_licks}

Hashbrown
