Robot Takeover

You know what it is, you know what to do… but can you handle the twist?

The site provided has a robots.txt file as show below.

Each time the page is refreshed a new robots.txt file will appear. Going to the disallow file(s) with the correct user agent will give one of two responses.

  1. HELLO ROBOT. UNFORTUNATELY I DO NOT HAVE A FLAG FOR YOU HERE.
  2. REJOICE, ROBOT. THE CHARACTER OF THE FLAG AT INDEX 8 IS THE SAME CHARACTER AT INDEX 3 IN THIS FILENAME.

The index varies per response.

Flag: flag{beep_boop_are_you_a_robot_too}
