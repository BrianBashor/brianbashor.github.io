Money Heist

Notes at Random:
2020-11-11
vuln hub

Money Heist is a web challenge box from vulnhub. Getting the flag involves finding a weak JSON web token (JWT) signature and editing the cookie. Afterwards you are provided with creds to access the server over ssh.

Initial scan

The nmap scan looks a lot busier than what’s actually going on. Basically, we got a web server and ssh running. In addition, port 53 (DNS), 3000 and 3001 are open.

Gobuster found a few things but nothing of interest.

JSON Web Token

Once we are logged in a JSON Web Token (JWT) is created.

Quick note on JWT:
A jwt consist of three base64 encoded strings seperated by a ..

  • Header
  • Payload
  • Verify Signature

Using jwt_tool we can brute force the signature.

Now that we have the signing key we can edit the jwt.

I changed the email from none@none.com to admin and added some time to the exp date.

 

 

 

We are going to use our newley discovered cerdentials to ssh into the box.

We got ASCII art so you know it’s legit.

Nothing further is needed to get the last flag. I’m not sure if the is an accidental permissions error.
